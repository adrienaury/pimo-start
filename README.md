# pimo-start

## Generators

| Nom     | Masque            | Cohérence | Réversible |
| ------- | ----------------- | --------- | ---------- |
| String  | regex             | + seed    | + cache    |
| Integer | randomInt         | + seed    | + cache    |
| Decimal | randomDecimal     | + seed    | + cache    |
| Date    | randDate          | + seed    | + cache    |
| List    | randomChoiceInUri | + seed    | + cache    |
| Custom  | template          | n/a       | n/a        |
